# .files

A modern Windows Explorer replacement built with Tauri, React, and Rust.

## Download

Head to the [Releases](https://github.com/200df7/dotfiles-releases/releases) page and download the latest installer.

## Features

- Multi-tab file browsing with split panes
- Built-in terminal (PowerShell)
- Git integration (stage, commit, push, pull, branch switching, diff viewer)
- Cloud drive support (OneDrive, Google Drive, GitHub repos)
- File preview (images, PDF, markdown, code, DOCX, XLSX, media)
- 9 built-in themes + custom theme editor with transparency/vibrancy
- System font picker with live preview
- Fuzzy search & command palette
- System tray with close-to-tray
- Windows context menu integration ("Open with .files")
- Single-instance with folder forwarding
- Drag & drop, inline rename, keyboard shortcuts

## Requirements

- Windows 10/11 (x64)
- WebView2 (included with Windows 11, auto-installed on Windows 10)
